담당: 커뮤니케이션팀 (정)이수정 선임 / (부)백채은 선임 / (부)하지훈 책임

## 구좌 1) 앱 메인 — Shopping info 영역

- 노출 위치: 앱 메인 Shopping info 영역 내 콘텐츠 카드
- 요청 납기 (게시희망일 기준, 본사 영업일):
  - 내용만 전달: **+3영업일 전**
  - 이미지 제작 필요: **+5영업일 전**
  - 페이지 제작 필요: **+10영업일 전**
- 각 점 자체 등록 진행 시에도 이미지 사이즈 등 아래 내용 준수 필요

## 구좌 3) 앱 메인팝업 — Daily 노출

- 노출 위치: 앱 메인 진입 시 팝업
- 요청 납기: 내용 전달 **+3영업일 전** / 이미지 제작 필요 시 **+5영업일 전**
- **(주의) 매일 노출되는 콘텐츠로 고객 피로감이 높은 구좌 — 최대 1주 노출 권장** (요청서 적색 강조)
- 노출대상: 자사 + H.Point 중 택 1 (전체 필요 시 자사 전체로 요청)

### 신청서 필드
게시시간 / 제목 / 지점 / 이동페이지 (utm_medium=app_popup) / 이미지 별도 전달 / 컨텐츠순서 / 노출대상

## 사이트 반영 방법
- "자사 매체 > APP" 매체 하위 지면(slots)으로 등재:
  - Shopping info 콘텐츠 — lead: "D-3 (이미지 제작 시 D-5, 페이지 제작 시 D-10 / 영업일)"
  - 메인팝업 (Daily) — lead: "D-3 (이미지 제작 시 D-5 / 영업일)", note: "고객 피로감 높은 구좌 — 최대 1주 노출 권장"
- 규격(px)은 신청서에 미기재 — 미디어키트의 APP 스펙(썸네일 메인 800×800, 하단 325×121,
  프리즘 750×750, PUSH 500×250)과 매핑 확인 필요. 구좌별 정확한 이미지 규격은
  이수정 선임/백채은 선임 확인 후 확정 (verified: false로 시작)
- UTM 규칙(utm_source=ehyundai_app, medium=app_main/app_popup)은 비고로 기재 —
  이동페이지 URL 작성 시 참조용
- 구좌 2)는 캡쳐에 없음 — 존재 여부·내용 확인 필요
