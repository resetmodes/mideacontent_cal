# 디벨롭 아이디어 백로그 ('26.7)

지금까지 만든 기능 위에서 더 키우면 좋을 것들. 우선순위·난이도·규빈 액션(대기 항목) 표기.
착수는 골라서 지시하면 됨. **[대기]** = 규빈이 자료/결정 주면 바로 착수 가능.

---

## 🔥 바로 이어서 (가치 높고 빠름)

1. **팀즈/이메일 알림 발송** — *진행 중.* 웹훅 URL만 확보되면 매일 아침 요약
   (오늘 촬영 · RMN 세금계산서/가부킹 전환 · 신규 등록 일정). **[대기: 웹훅 URL]**
2. **RMN 미수금 aging** — 정산 요약에 30/60/90일 경과 미수금 분리 표시 + 월말 알림.
   지금은 미수금 총액만 — 오래된 미수금이 안 보임. (세금계산서 발행일·입금일 필드 추가)
3. **홈 화면에 RMN 요약 카드** — 이번 달 매출·미수금·마감 임박 건을 홈 상단에.
   지금 홈은 근태·콘텐츠 중심 — 영업 현황이 안 보임. (데이터 이미 있음, 카드만)
4. **소재 규격 자가 검수** — 제작한 이미지 파일을 드롭하면 사이즈·용량을 매체 스펙과
   자동 대조("650×242 맞음 / 1MB 초과"). 브라우저 처리, 무료. 대행사 반려 사고 예방.
5. **매체 스펙 → 대행사 전달 PDF** — 선택한 매체 지면 스펙만 묶어 원클릭 PDF.
   지금은 링크 공유만 — 오프라인 전달·첨부용 PDF 수요.

## 📈 중기 (가치 높고 중간 난이도)

6. **RMN 대시보드** — 월별 매출 추이 그래프 · 상품별 판매 비중 · 판매사별 실적 ·
   광고주 재구매율. 지금은 표만 — 추이·비중이 한눈에 안 들어옴.
7. **RMN 재고 히트맵** — 상품별 잔여 구좌를 기간 히트맵으로("9월 둘째 주 스플래시 비었음")
   → 역으로 영업 제안 근거. 재고 로직(slotAvailability) 이미 있음, 시각화만.
8. **광고주 미니 CRM** — 광고주별 과거 집행 이력·총액·담당·마지막 접촉 → 재접촉 시점 알림.
   RMN 데이터에 이미 광고주별 히스토리 존재 — 뷰만 추가.
9. **정기 일정 반복 등록** — 유튜브 라인업(미팅룸 주1회 등)·정기 회의처럼 반복되는 건을
   한 번 등록 = 자동 반복 (기념일 반복 로직 재사용). 매번 손등록 제거.
10. **캘린더 구독(iCal) 내보내기** — 팀원이 개인 구글/아웃룩 캘린더에서 팀 일정을 구독.
    "사이트 안 봐도 내 캘린더에 뜸" — 확산·정착에 큰 도움.
11. **휴점일 정기 규칙 자동 생성** — "매월 N주 월요일" 규칙 넣으면 연간 자동 채움
    (지금은 한 줄씩 수기). 규칙만 알려주면 됨. **[대기: 휴점 규칙]**

## 🧭 큰 그림 (임팩트 크나 준비 필요)

12. **GA4 연동 (RMN 3차)** — 노출·클릭 자동 수집 → 부킹 첨부 + 부쉐론 결과보고서 자동 생성.
    출력 단계·계획 문서 완비. **[대기: GA 관리자 권한 + JSON 키]**
13. **월간 집행 리포트** — 데이터 계층(report.js)·계약 문서 완비. 양식 오면 매핑만.
    **[대기: 규빈 양식]**
14. **임원 보고용 원페이지** — 모니터링 + RMN + 일정 핵심을 한 장 HTML 리포트로 자동 생성
    (주간/월간). exec-report-design 기준 적용.
15. **타겟형·서브계정 스펙 목업 채우기** — 아파트너·키즈노트 등 + 도시매뉴얼·룸넘버에
    인스타처럼 캡처/목업·포지셔닝 채움. **[대기: 각 계정 캡처·소개]**
16. **판매사 마스터 완성** — 나스미디어 등 4사 사업자번호·주소 → 청약서 자동완성 완결.
    **[대기: 4사 정보]**
17. **AI 어시스턴트** — 설계 확정(ai-assistant-design.md), 스캐폴드 존재. "이번 달 샤넬
    집행 얼마였지?" 같은 자연어 질의. Opus 세션 예약.

## 규빈 액션이면 바로 되는 것 (자료·결정만)

- 팀즈 웹훅 URL → 알림 발송 (#1)
- GA 관리자 권한 + 키 → RMN 노출·클릭 자동화 (#12)
- 판매사 4사 정보 → 청약서 완결 (#16)
- 각 채널 캡처·소개 → 스펙 목업 채움 (#15)
- 월간 리포트 양식 → 리포트 착수 (#13)
- 휴점 정기 규칙 → 자동 생성 (#11)

## 추천 다음 3개 (내 의견)

**#1 알림(진행 중) → #3 홈 RMN 요약 카드 → #6 RMN 대시보드.**
셋 다 데이터가 이미 있어 빠르고, "매일 열어보는 화면에서 영업·마감이 한눈에"라는
공통 효과. 알림으로 밀고(push), 홈·대시보드로 당기는(pull) 조합.
