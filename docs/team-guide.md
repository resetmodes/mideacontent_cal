# 미디어콘텐츠팀 포털 사용 가이드 — 팀원용 1장

> 사이트: https://mediacontent-cal.vercel.app (팀 계정 로그인 필요)
> 링크 공유 규칙은 맨 아래 "누구에게 어떤 링크?" 를 꼭 지켜주세요.

## 화면 구성 (탭)

접속하면 **홈**이 첫 화면입니다. 상단 탭으로 이동해요.

| 탭 | 용도 |
|---|---|
| **홈** | 오늘 한눈에 — 근태·주요 콘텐츠·이번 주 촬영·채널 이슈·하이라이트 |
| **팀 일정** | 연차·반차·외근·출장·교육·업무·기념일 (팀 내부용, 미러 미표시) |
| **매체 캘린더** | 광고 집행(업로드·발행) 일정 — 팀 공유 |
| **촬영일정** | 유튜브·인스타 촬영 스케줄 |
| **매체 스펙** | 전 매체 소재 규격·납기·프로세스·담당 |
| **매체 모니터링** | 자사·경쟁사 SNS 지표 + UGC + 타겟APP 실적 |

## 홈 화면

접속 첫 화면이자 팀 업무 요약판입니다. 데이터가 없는 섹션은 자동으로 숨겨져요.

- 상단 지표 4개(게시 예정·캠페인·촬영·부재)는 **클릭하면 해당 탭으로 이동**
- ① 오늘·내일 팀원 근태 → ② 주요 콘텐츠 D-day(캠페인) → ③ 이번 주 촬영
  → ④ **채널 이슈**(팔로워 급변·휴면·조회 급등) → ⑤ 이번 주 하이라이트(유튜브 썸네일)
- **기념일 축하 팝업** — 팀원 생일 등 기념일에 홈 접속 시 뜹니다.
  주말·공휴일이면 **직전 영업일에 미리** 떠요 (일요일 → 금요일). 하루 한 번만.

## 일정 등록 — 한 줄이면 끝

캘린더 상단 입력창에 치고 Enter. 날짜·매체·캠페인을 알아서 인식합니다.
**매체 캘린더·촬영일정·팀 일정 모두 같은 방식**입니다.

| 이렇게 치면 | 이렇게 등록됩니다 |
|---|---|
| `12/20 크리스마스 릴스 인스타 #크리스마스` | 12/20 · 인스타 · #크리스마스 캠페인 |
| `12/20~25 연말 팝업 카톡` | 12/20~25 기간 일정 |
| `7월 10일 도시 신규 콘텐츠` | 도시메뉴얼 계정으로 자동 분류 ("도메"도 인식) |
| `내일 앱푸쉬 발송` | 내일 날짜 · 백화점APP 푸쉬 |
| `다음주 목요일 앱푸쉬` | **상대 날짜도 인식** — 다다음주 월요일 · 지난주 토요일 · 차주 화요일 등 (주 시작 = 월요일) |
| `다음주 월~수 김상수 출장` | **상대 날짜 기간** — 다음주 월요일~수요일 3일짜리 일정 |
| `12/20 성탄 티저 인스타+유튜브+카톡` | **3개 매체에 동시 등록** (+로 연결) |
| `7/25 여름 세일 바이비+아파트너+카카오골프` | **타겟APP 세부 3종 동시 등록** — 캘린더에는 "×3" 한 줄로 묶여 표시 |
| `7/10 촬영 7/15 업로드 여름 룩북 인스타` | **촬영 건은 촬영일정 탭, 업로드 건은 매체 캘린더에 각각** |

- 입력하면 아래 **상태 칩**으로 뭐가 인식됐는지 미리 보입니다 — 빨간 점선 칩은 미인식(등록 시 선택 팝업)
- `#`만 치면 기존 캠페인 목록이 떠서 이름을 통일할 수 있어요 (최근 2개월 내 게시된 캠페인만 제안)
- 날짜 칸을 직접 클릭해서 폼으로 등록해도 됩니다 (모달 안 "한 줄 자동 작성"도 지원)
- **타겟APP 여러 매체 동시 집행**: 날짜 클릭 → 매체 "타겟 APP" 선택 → 세부를 **체크박스로
  여러 개 선택**(전체 선택 버튼 있음) → 체크한 수만큼 한 번에 등록. 캘린더 셀에는
  "×N" 한 줄로 묶여 보이고, 클릭하면 세부별 상세로 들어갈 수 있어요
- **휴대폰에서는** 날짜 클릭 시 화면 위쪽에 작은 시트가 떠요 — 한 줄로 치고 등록하면 끝.
  매체·기간·메모를 직접 고르려면 "상세 입력 펼치기"를 누르세요
- **작성자는 자동** — 로그인 계정으로 기록됩니다 (따로 쓸 필요 없음)

### 팀 일정 (연차·기념일 등)

- 같은 한 줄 문법: `7/20 노규빈 연차`, `12/5 노규빈 생일`
- **내 일정은 이름 생략 가능** — `7/16 연차`라고만 써도 로그인 계정 이름이 자동으로 붙어
  캘린더에 "노규빈 연차"로 표시됩니다
- 남의 일정을 대신 등록할 땐 이름을 병기하세요 — 이름이 들어 있으면 쓴 그대로 표시됩니다.
  예) `김상수, 정소미, 노규빈 목동 외근` → 그대로 캘린더에 표시
- 유형별 색: 연차 빨강 · 반차 주황 · 외근 파랑 · 출장/교육 · **업무(그린)** · 기념일(무지개)
- **업무 일정**: `8/1 임원 보고 자료 마감`처럼 마감·회의·보고 키워드로 등록 — 홈 화면
  "주요 업무·마감" 섹션에 D-day로 떠서 팀 전체가 볼 수 있어요 (이름은 자동으로 안 붙음)
- **기념일은 1건만 등록하면 매년 자동 반복**됩니다 (다시 등록할 필요 없음)

## 일정 확인·수정

- **날짜를 잘못 등록했다면 드래그로 이동** — 일정을 끌어서 다른 날짜 칸에 놓으면 이동됩니다
  (기간 일정은 통째로 이동). 실수로 옮겼으면 하단의 **[실행 취소]**를 8초 안에 누르세요.
  PC 전용 — 휴대폰에서는 일정 클릭 → 수정 폼에서 날짜를 바꾸세요
- **날짜 숫자를 클릭하면 그날 일정 전체**가 큰 화면으로 떠요 (셀에 "+N 더보기"가 있을 때도
  클릭) — 타겟APP처럼 하루에 여러 건일 때 여기서 다 볼 수 있습니다. 미러 사이트에서도 동일
- 일정 클릭 → 상세 팝업: 수정·삭제(두 번 클릭 확인)·메모(링크는 클릭 가능)
- 팝업 안 **"이 매체 규격·납기 보기 →"** — 그 매체의 소재 규격·마감으로 바로 이동
- **집행 실적 후보** — 인스타·유튜브 일정엔 실제 게시물이 자동 매칭됩니다. 맞는 걸
  **[선택]** 하면 그것만 확정으로 남아요 (보고용 데이터가 됩니다 — 담당 건은 꼭 선택!)
- 상단 검색창(그레이 "검색" 라벨·밑줄 입력): 제목·캠페인·메모·작성자·매체로 전체 일정 검색 —
  그린 "등록" 라벨이 붙은 박스 입력칸은 등록용이니 헷갈리지 마세요
- 변경 이력: 팝업의 "변경 이력" / 캘린더 하단 "최근 30일 삭제 기록"
- 캠페인 뷰: 최근 1개월 내 게시된 캠페인은 펼쳐지고, 오래된 캠페인은 접혀 보관됩니다

## 매체 모니터링

- 인스타그램 / 유튜브 / **UGC** / **타겟APP** 세그먼트 전환
- 상단 **하이라이트** = 자동 브리핑 (팔로워 급변·새 휴면·조회 급등) — 초록 강조 박스
- UGC = 고객·인플루언서가 해시태그(#더현대서울 등)로 올린 게시물의 감정·주제·주간 동향
- 팔로워·구독자 옆 ▲▼ = 직전 수집 대비 증감
- SNS 수집은 매주 자동 · 타겟APP 실적은 매월 초 전월분 수기 입력 (담당: 규빈)

## ⚠ 입력 시 보안 수칙 (중요)

- 일정 **제목·메모에 단가·계약금액·개인 연락처·미공개 사업 정보를 쓰지 마세요.**
  타 팀 공유용 미러 사이트는 로그인이 없어서, 주소를 아는 사람은 일정을 볼 수 있습니다
- 지점 사고 사례를 적을 땐 "모 지점"으로 익명화

## 누구에게 어떤 링크? (공유 규칙 — 반드시 준수)

| 대상 | 주는 것 | 절대 금지 |
|---|---|---|
| **우리 팀원** | 본 사이트 주소 + 개인 계정 (발급: 규빈) | 계정 공유 금지 |
| **사내 타 팀** | **미러 사이트 주소** (로그인 없음, 읽기 전용) | 본 사이트 계정 발급 금지 |
| **대행사·지점 (외부)** | 스펙 탭에서 해당 매체 열고 **"이 매체 외부 링크 복사"** 버튼 → 그 링크만 전달 (로그인 없음, 담당자명·내부 지표 자동 숨김, 캘린더 없음) | **외부인에게 어떤 계정도 발급 금지** · 미러 주소 전달 금지 |

- 미러 주소는 사내 채널로만 전달 (외부 메일·단톡 금지) — 주소 자체가 열쇠입니다
- 팀 일정·매체 모니터링은 미러/외부에 **표시되지 않습니다** (내부 전용)
- 링크가 외부에 퍼진 것 같으면 → 즉시 규빈에게 (회수 절차는 운영 런북 참조)
