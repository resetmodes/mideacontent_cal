# CLAUDE.md — 매체 스펙 라이브러리 + 매체 일정 캘린더 (media-spec-library)

> **작업 전 필독: `PLAYBOOK.md`** — 작업 절차·금지선·파일 지도·검증 도구('26.7 하네스).
> 코드 수정 후에는 반드시 `npm run verify` (파서·가드 테스트 + 빌드) 통과 확인.

## 프로젝트 목적
현대백화점 미디어콘텐츠팀의 매체 운영 허브. 상단 탭 2개:
1. **매체 스펙** — 전 매체의 소재 규격, 납기, 진행 프로세스, 심의·금지사항, 담당자 조회.
   지점 담당자·대행사에 링크 하나로 전달. 등록·수정은 데이터 파일 직접 편집.
2. **매체 캘린더** ('26.7 추가) — 팀 내 광고 집행 일정 공유. 빠른 입력 한 줄로 등록
   ("12/20 크리스마스 인스타 릴스 #크리스마스"), 클릭해서 수정·삭제, 캠페인(#태그)별 묶어 보기.

  지표 집계 1개월, 휴면 = 30일+ 미게시. 최근 1개월 게시 0건인 계정은 raw가 비는데,
  clean-instagram.mjs가 지표 없이 "휴면"으로 계정을 유지함 (대시보드에서 사라지지 않게)
- **비용 ('26.7)**: Apify는 결과 1건당 과금 — 인스타 $2.70/1,000건(무료 플랜), 유튜브 $0.50/1,000건.
  1개월 윈도우 기준 1회 수집 ≈ 인스타 ~600건($1.6) + 유튜브 151건($0.08) ≈ **$1.7/회**.
  주 1회면 월 ~$7.4로 무료 크레딧($5/월) 초과 — 무료 한도 내로 쓰려면 격주 수집(월 ~$3.7).
  격주 cron: `'0 0 1-7,15-21 * 1'` (매월 1~7일·15~21일의 월요일) — **'26.7 격주로 전환 적용됨**

  이미 발생했는데 데이터는 저장 안 된 상태. sns-collect.yml의 push 재시도(rebase 3회)로

## 도메인 지식 (혼동 주의)
- **APT LCD ≠ 아파트앱**: APT LCD는 엘리베이터 영상 매체(포커스미디어/KT타운보드),

- 예산·비용 표기는 천 원 단위 (40,000천 원)
- 지점 사고 사례 언급 시 "모 지점" 익명화
- 주류 관련: APT LCD는 주류(와인) 이미지 게첨 불가 / 데일리샷·Meta는 표현 검수 필수
- 사내 파일은 DRM(DRM-ONE) 걸려 있는 경우 많음 — 파싱 실패 시 사용자에게 해제본 요청
